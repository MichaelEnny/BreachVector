## Capture tips

- Use seeded showcase data for the cleanest report screenshots.
- Hide browser bookmarks and unrelated tabs.
- Prefer one consistent theme and zoom level for all captures.
- If you include a live domain, avoid using any target that might create an awkward score during the demo.
